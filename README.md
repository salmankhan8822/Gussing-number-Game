# Number Guessing Game
A simple and interactive Number Guessing Game built using **HTML, CSS, and JavaScript**.
The computer generates a random number between **1 and 10**, and the user has to guess the correct number.
---
# Features
- Random number generation using `Math.random()`
- User input validation
- Instant feedback (Too high / Too low / Correct)
- Restart functionality
- Beginner-friendly JavaScript logic
---
# Technologies Used
- HTML
- CSS
- JavaScript (DOM Manipulation & Event Handling)
---
# How It Works

1. The computer generates a random number between 1 and 10.
2. The user enters a number in the input field.
3. The game checks:
   - If the guess is correct
   - If the number is too high
   - If the number is too low
   - If the input is invalid
4. The restart button resets the game message.
